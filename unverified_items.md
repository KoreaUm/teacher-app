# 검증 불가 / 추가 확인 필요 목록

확인일: 2026-05-05 (2차 업데이트)

## 전수조사 범위

- 최근 3년 전체 공공기관 고졸 전형 전수조사는 아직 완료되지 않았다.
- 현재 데이터셋은 공식 ALIO/PDF에서 확인한 초기 시드 데이터이며, 추천 로직에서는 `status`가 `검증됨`인 항목만 강하게 반영해야 한다.
- `부분확인`, `확인불가`, `만료/과거공고`는 사용자에게 근거 안내용으로만 표시하고 자동 추천 점수에는 넣지 않는 것이 안전하다.

## 데이터 정제 이슈

- 기관별로 `자격증 가산점`, `응시자격`, `서류 점수`, `정규직 전환 조건`이 섞여 있으므로 하나의 점수 체계로 합산하면 안 된다.
- `고졸`, `학력무관`, `고졸인재`, `보훈 고졸`, `장애인 고졸`, `체험형 인턴`은 서로 다른 추천 트랙으로 분리해야 한다.
- 원문에 점수는 있으나 자격증명이 포괄 표현인 경우, 자격증명 매칭 추천에는 사용하지 않는다.
- 내신/성적 반영 여부는 현재 확인된 공고에서 대부분 명시적 점수표를 확보하지 못했다. `확인불가`를 기본값으로 유지한다.
- 상업고 금융권 추천에서는 `기업회계`, `전산회계`, `전산세무`, `컴퓨터활용능력`, `한국사`, `정보처리`가 자주 등장할 수 있으나, 기관별 공식 공고에서 점수와 전형단계가 확인된 경우에만 점수화한다.
