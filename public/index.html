주요 기능

할일 관리, 학사 일정, 시간표, 학급 기록, 상담 기록, 성적/취업 자료 관리 등 교사업무에 필요한 기능을 제공합니다.

Google 연동

사용자가 동의한 경우에만 Google Tasks와 Google Calendar에 할일과 개인 일정을 동기화합니다.

데이터 보호

학생 개인정보, 상담 기록, 성적 데이터 등 민감한 자료는 외부 Google 서비스로 자동 전송하지 않도록 설계합니다.

Google API 사용 안내

쌤포트는 Google API를 사용하여 사용자가 직접 등록한 할일과 일정을 사용자의 Google 계정에 저장하거나 갱신합니다. Google 사용자 데이터의 사용은 사용자가 동의한 기능 제공 목적에 한정됩니다.
