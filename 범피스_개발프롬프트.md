## 1. 마크다운 파싱 파이프라인

선택된 한글 문서 텍스트를 파싱하는 과정:

```
블록텍스트()
  → split('\r\n')             # 줄 분리
  → 콜론처리()                # ':' 없는 줄 → '엔터:엔터'
  → [s.split(':', 1) ...]    # 키워드:내용 분리
  → 엔터처리()                # [''] → ['엔터','엔터']
  → 키워드처리()              # #오늘날짜 → YYYYMMDD
  → 특수문자제거()            # 키워드에서 한글/공백 외 제거
  → 배열공백제거()            # strip()
```

### Python 구현

```python
def 콜론처리(배열):
    결과 = []
    for 아이템 in 배열:
        if ':' not in 아이템:
            결과.append('엔터:엔터')
        else:
            결과.append(아이템)
    return 결과

def 엔터처리(배열):
    결과 = []
    for 아이템 in 배열:
        if 아이템 == ['']:
            결과.append(['엔터', '엔터'])
        else:
            결과.append(아이템)
    return 결과

def 키워드처리(배열):
    from datetime import datetime
    return [
        [s.replace('#오늘날짜', datetime.today().strftime('%Y%m%d')) for s in inner]
        for inner in 배열
    ]

def 특수문자제거(배열):
    import re
    pat = re.compile(r'[^가-힣\s]')
    return [[pat.sub('', item[0]), item[1]] for item in 배열]

def 배열공백제거(배열):
    return [[s.strip() for s in inner] for inner in 배열]

def 임시변환기(텍스트):
    """원1~원10 → ①②...⑩"""
    for i in range(1, 11):
        텍스트 = 텍스트.replace(f'원{i}', chr(9311 + i))
    return 텍스트
```

## 4. 표 처리 로직

```
표:항목:내용:비고          ← 키: '표', 값: '항목:내용:비고'
구분:구분                  ← 표 안에서 구분행
소제목:1단계 목표          ← 표 내용 행
하:                        ← 빈 행
```

처리 흐름:
1. `표:컬럼1:컬럼2:...` → `표진입=1`, `표개수=컬럼수`
2. 이후 키가 `표`이면서 `표진입==1` → `re.split(':', 값)`으로 셀 분리
3. `self.표오른쪽(1)` 이동하며 각 셀에 `self.문장()` 입력
4. 다른 키 등장 시 → `MovePos(105, 107, 23)` + `self.표나가기()` 호출

## 5. 소제목 너비 계산식

```python
너비 = 4 + 6.2 * len(내용) - 2.9 * 내용.count(' ')
```

이 값(단위: mm 또는 HWP 내부 단위)을 `self.소제목(텍스트, 너비, 2.9)` 의 두 번째 인자로 전달합니다.

## 7. 순화검색 기능

선택된 단어를 `단어순화1` 사전에서 검색하여 순화어 제안.

```python
def 순화검색(텍스트):
    공백수 = 텍스트.count(' ')
    if 공백수 == 0:
        딕셔너리 = 단어순화1   # 197개
    elif 공백수 == 1:
        딕셔너리 = 단어순화2
    else:
        딕셔너리 = 단어순화3
    
    if 텍스트 in 딕셔너리:
        결과 = 딕셔너리[텍스트]
        return [텍스트, 결과[0], 결과[1]]  # [원문, 추천, 설명]
    return None
```

주요 외래어 순화어 예시:
- `워크숍` → `공동 연수` (연수/연구 모임)
- `인프라` → `기반시설`
- `로드맵` → `(단계별) 이행안/단계별 계획`
- `벤치마킹` → `참고 연구/사례 연구`
- `모니터링` → `점검/관찰`
- `바우처` → `이용권`
- `네트워크` → `연결망/관계망/연계망`
- `노하우` → `비법/기술/비결/방법`
- `비전` → `미래상/큰 꿈/청사진`
- `콘텐츠` → `내용물` (원본에서는 `컨텐츠` → `콘텐츠` 오타 수정도 포함)
